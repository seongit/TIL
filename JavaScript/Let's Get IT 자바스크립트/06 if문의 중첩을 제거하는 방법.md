### if문의 중첩을 제거하는 방법
1. if문 다음에 나오는 공통된 절차를 각 분기점 내부에 넣는다.
2. 분기점에서 짧은 절차부터 실행하게 if문을 작성한다.
3. 짧은 절차가 끝나면 return이나 break으로 중단한다.
4. else를 제거한다
5. 다음 중첩된 분기점이 나오면 1~4의 과정을 반복한다.

        ;✔ 같은 위치의 내용을 지울 때 휠을 눌러서 드래그하면 편하다.
