#### 소수 구하는 방법
소수는 1과 자기자신만 있는 수이다.
이 말인 즉, 1과 자기자신을 제외하고 나누어 떨어지는 수는 소수가 아니라는 뜻이다.

소수 : 5 -> 1 , 5
소수가 아님 : 6 -> 1 , 2 , 3 , 6

따라서 조건식은 다음과 같다.

```java 
for(int i=1;i<n;i++){
    // cnt가 1 이상이면 소수가 아니다.
    if(n%i==0) {
        cnt++;
    }
}
```
