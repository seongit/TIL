
## 리플렉션이란
리플렉션은 힙 영역에 로드 된 Class 타입의 객체를 통해, 접근 제어자 상관없이 원하는 클래스의 정보에 접근해서 조작할 수 있도록 지원하는 API이다.
클래스의 메타 정보를 기반으로 클래스에 정의된 메서드, 필드, 생성자등을 조회하고 이를 통해 객체 인스턴스를 생성하거나 메서드를 호출할 수 있음 → 이런 작업을 리플렉션이라고 함!
